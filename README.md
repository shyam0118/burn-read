# Burn Read — Get Roasted by AI 🔥

**Burn Read** is a privacy-first, browser-based WhatsApp chat analyzer. Upload your chat export, see detailed analytics, beautiful visualizations, and get hilariously roasted by AI based on your messaging habits.

> **Core Principle:** Everything runs in your browser. Your chat data never leaves your device. No server, no database, no logs.

## ✨ Features

- **100% Private:** Parsing and analysis happen locally in your browser using JavaScript.
- **Deep Analytics:** Total messages, word counts, active hours, response times, and ghost streaks.
- **Visualizations:**
  - Message share (Pie chart)
  - Activity over time (Line chart)
  - Heatmap (Hour vs Day)
  - Word Cloud (Top used words)
- **AI Roasts:** Get personalized, hilarious roasts generated by AI (powered by OpenRouter/Gemini).
  - 5 Tones: Savage, Friendly, Bollywood, Cricket, and Corporate.
- **Personality Tags:** Fun auto-assigned labels like "Night Owl", "Novelist", and "Ghost".
- **Shareable Cards:** Download Instagram-ready cards with your stats and roasts.

## 🚀 Setup & Local Development

Follow these steps to get the project running on your local machine.

### Prerequisites

- **Node.js 18.x** or higher
- **npm** or **pnpm**
- An **OpenRouter API Key** (for AI roasts)

### 1. Clone the repository

```bash
git clone https://github.com/shyam0118/burn-read.git
cd burn-read
```

### 2. Install dependencies

```bash
npm install
```

### 3. Environment Variables

Create a `.env` file in the root directory and add your OpenRouter API key:

```env
OPENROUTER_API_KEY=your_api_key_here
```

*Optional:* You can also specify a custom model or Turnstile keys:
```env
# Optional
OPENROUTER_MODEL=google/gemini-2.0-flash-exp:free
NEXT_PUBLIC_TURNSTILE_SITE_KEY=your_site_key
TURNSTILE_SECRET_KEY=your_secret_key
```

### 4. Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### 5. Build for production

```bash
npm run build
npm start
```
